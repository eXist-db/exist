oNVDL and Jing

oNVDL is developed on top of Jing to add NVDL support. The Jing version modified by oNVDL is Jing version 20030619 covered by the conditions found in jing.LICENSE.txt.

The changes in oNVDL compared with Jing version 20030619 are:

    * Added NVDL support.
    * Updated the support for Xerces to work with Xerces 2.8.0.
    * Updated Saxon to Saxon 6.5.5.
    * Handle correctly the XML Schema pattern facet.
    * Removed the Xerces J 1 Regular Expression Engine implementation (com/thaiopensource/datatype/xsd/regex/xerces/RegexEngineImpl.java).
