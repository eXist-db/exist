Do you want to be "XRX Ready" ? Just add a stylesheet at the beginning of your XForms XML document !

XSLTForms can convert XForms to XHTML+Javascript (AJAX) on any browser or web-server (PHP, ASP, J2EE,...) with an XSLT 1.0 engine.

It works fine with Internet Explorer, Mozilla Firefox and Opera and almost fine with Safari and Chrome (small bugs to be fixed).

This is an opensource project with a LGPL licence.

It includes XSLT parts and Javascript parts from the AJAXForms Java project (not supported since 2006).

Current colored W3C XForms Test Suite : http://www.agencexml.com/xforms-tests/testsuite/XForms1.1/Edition1/driverPages/html/
