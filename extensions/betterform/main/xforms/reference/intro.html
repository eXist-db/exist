XForms Feature Explorer

Welcome to the XForms Feature Explorer!

The XFormsFeatureExplorer is the central location for the betterFORM live documentation. It will be constantly updated and extended when new features become available.

Every page has a short description along with the link to the XForms 1.1 REC where appropriate. Likewise each component has its own description and optional notes attached.

The code snippets show the markup for the live sample and can be used directly in your own forms.
