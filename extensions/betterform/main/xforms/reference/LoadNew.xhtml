Load element with load="new"

This page demonstrates the use of a XForms load element with load new.

Example

Load this Page in a new window

XForms Markup

<xf:trigger>
    <xf:label>Load this Page in a new window</xf:label>
    <xf:action>
        <xf:load resource="LoadNew.xhtml" show="new"></xf:load>
    </xf:action>
</xf:trigger>
