select control allowing users to type in a value

This form control allows the user to make multiple selections from a set of choices.

A select using attribute selection with value 'open' allows a user to input a value that is not available in the predefined list of values.

the selection attribute takes precedence of the appearance attribute. The rendered control will always be a dropdown.
