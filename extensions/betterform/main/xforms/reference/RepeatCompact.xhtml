    <xf:input ref="item[1]">
        <xf:label>String:</xf:label>
    </xf:input>
    <xf:input ref="item[2]">
        <xf:label>Date:</xf:label>
    </xf:input>
    <xf:input ref="item[3]">
        <xf:label>Boolean:</xf:label>
    </xf:input>
</xf:repeat>

Notes

The CSS class 'bfTableCol-[number of column] is automatically generated for every column in the compact repeat.
