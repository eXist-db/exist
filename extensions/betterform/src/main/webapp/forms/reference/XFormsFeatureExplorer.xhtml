XForms Feature Explorer

Welcome to the XForms Feature Explorer!

The XFormsFeatureExplorer is the central for the betterFORM live documentation. It will be constantly updated and extended when new features become available.

Every page has a short description along with the link to the XForms 1.1 REC where appropriate. Likewise each component has its own description and optional notes attached.

The code snippets show the markup for the live sample and can be used directly in your own forms. You can expand and collapse all sections by clicking on the header bar to organize your view.

Features of the FeatureExplorer

The XFormsFeatureExplorer itself uses some techniques that are unique to betterFORM:

  • it uses embedding to dynamically integrate XHTML/XForms documents
  • it demonstrates the use of the Dojo Tabcontainer as a XForms switch
  • it uses mixed-in Dojo components like the TitlePane to layout complex content
