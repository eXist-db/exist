Launching eXist-db

On Windows and Linux, use the eXist-db start menu item for launching eXist-db. If you have installed a desktop icon, you can also double-click this to launch eXist-db. On Mac OS, double-click the eXist-db.app icon inside the folder in which eXist-db is installed.

You can also launch eXist-db by double-clicking on start.jar or by calling "java -jar start.jar" on a command line.

Launching eXist-db in one of these ways will display a splash screen while starting up the database. After startup, you will find an eXist-db icon in your system tray/menu bar. Click (Windows and Mac OS) or right-click (Linux) on the icon to get a popup menu with further options.

Launching a Headless System

If you wish to run a headless system, use the shell scripts supplied in the bin/ directory to launch eXist-db.

Accessing the Server

Once the server is running, you can browse to

http://localhost:8080/exist/apps/doc/

to view the documentation. We suggest you read the Quick Start Guide first.

Shutting Down the Server

You can shut down eXist-db from the system tray/menu bar icon or from the Dashboard at http://localhost:8080/exist/apps/dashboard/.
