Using the Launcher

On Windows or Linux use the start menu entry for launching eXist-db. On Mac OS, double click the eXist.app icon. This will show a splash screen while starting up the database. After startup, you should find a small eXist-db icon in your system tray. Right click on it to get a popup menu with further options.

If you haven't installed eXist-db into the start menu, you may also double click on start.jar or call "java -jar start.jar" on a command line.

Using Shell Scripts

If you are running on a headless system, use the supplied shell scripts in the bin/ directory to start eXist-db.

Accessing the Server

Once the server is running, you can browse to

http://localhost:8080/exist/

to view the documentation. Read the Quick Start Guide first.
